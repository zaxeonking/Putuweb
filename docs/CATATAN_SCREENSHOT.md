# Screenshots

Folder ini berisi gambar untuk README.md.

`screenshot-dashboard.png` saat ini masih berupa **placeholder** (bukan
tangkapan layar sungguhan). Setelah menjalankan aplikasi (`npm run dev`),
ambil screenshot halaman-halaman berikut dan timpa/tambahkan filenya di
sini agar README menampilkan tampilan asli:

- `screenshot-dashboard.png` — halaman Beranda (`/`)
- `screenshot-kelas.png` — halaman Kelas (`/kelas`)
- `screenshot-jadwal.png` — halaman Jadwal (`/jadwal`)
- `screenshot-mobile.png` — tampilan mobile (sidebar sebagai drawer)

Lalu tambahkan referensinya di README.md kalau menambahkan gambar baru:

```markdown
![Halaman Kelas](docs/screenshot-kelas.png)
```
